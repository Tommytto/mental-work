# Sliding Window Patterns

| Pattern | Function | When to Use |
|---------|----------|-------------|
| Fixed window sum | `MaxSumOfArrayWindow` | Max/min sum of contiguous subarray of size K |
| Fixed window count | `MaxVowels` | Count property in every window of size K |
| Expand/shrink (hashmap) | `LongestSubNonRep` | Longest substring without repeating characters |
| Monotonic deque | `SlidingWindowMax` | Max/min in every sliding window of size K |
| Shrink when condition met | `MinSubarrayLenAtLeast` | Shortest subarray with sum >= S |
| At most K distinct | `AtMostKDistinct` | Count subarrays with at most K distinct elements |
| Exactly K distinct | `SubarraysWithExactlyKDistinct` | Use atMost(K) - atMost(K-1) trick |
| Frequency matching | `FindAnagrams` | Find all anagram positions of pattern in string |
